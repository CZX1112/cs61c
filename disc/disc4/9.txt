将 RISC-V 代码转换为 C。这个 RISC-V 函数计算是什么？ 假设没有堆栈或与内存相关的问题，
并且假设没有负输入

// a0 -> x, a1 -> y,
// t0 -> result
// Function computes pow(x, y)
// Direct translation:
int power(int x, int y) {
    int result = 1;
    while (y != 0) {
        result *= x;
        y--;
    }
    return result;
}
